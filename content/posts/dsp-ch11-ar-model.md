## The Problem

When estimating the frequency of a finite-length sinusoidal signal, the intuitive approach — find the largest DFT/DTFT component — has a fundamental limitation: **resolution is bounded by the window length**.

For two closely-spaced sinusoids, if their frequency separation is smaller than the window's main lobe width, they cannot be resolved by conventional methods.

## The Solution: Parametric Signal Modeling (AR Model)

Professor Oppenheim introduced an alternative: **Parametric Signal Modeling**, also known as the Autoregressive (AR) Model.

Instead of assuming nothing about signal structure, we assume the signal was generated by an all-pole system:

$$H(z) = \frac{1}{1 + \sum_{k=1}^{p} a_k z^{-k}}$$

We then estimate the system coefficients from observations, and use the system's DTFT as our power spectrum estimate.

## Two Estimation Methods

### Autocorrelation Method (`aryule`)

- Uses all M+p length prediction error
- Applies Hamming window to reduce edge effects
- **Guarantees system stability**
- Cannot entirely eliminate end-effect errors

### Covariance Method (`arcov`)

- Only considers error in the interval [p, M]
- Eliminates end effects without windowing
- Can achieve zero prediction error for deterministic signals
- **Does not guarantee stability**

## Practical Example

Two closely-spaced sinusoids with only 101 samples:

$$s[n] = 20\cos(0.2\pi n - 0.1\pi) + 22\cos(0.22\pi n + 0.9\pi)$$

The frequency separation is 0.02π, but the window main lobe is 0.04π — conventional DFT and autocorrelation methods **fail to resolve** these components. Only the covariance method succeeds.

## MATLAB Code

```matlab
n = 0:101;
s = 20*cos(0.2*pi*n - 0.1*pi) + 22*cos(0.22*pi*n + 0.9*pi);

[d, p] = aryule(s, 4);   % autocorrelation method
[a, e] = arcov(s, 4);    % covariance method

[Ha, wa] = freqz(sqrt(p), d);
[Hc, wc] = freqz(sqrt(e), a);

periodogram(s)
hold on;
plot(wa/pi, 20*log10(2*abs(Ha)/(2*pi)), 'r', 'LineWidth', 2);
plot(wc/pi, 20*log10(2*abs(Hc)/(2*pi)), 'b', 'LineWidth', 2);
xlabel('Normalized frequency (\times\pi rad/sample)');
ylabel('One-sided PSD (dB/rad/sample)');
legend('Periodogram', 'AR (autocorrelation)', 'AR (covariance)');
```

## Key Takeaway

The AR model trades a structural assumption (all-pole system) for dramatically improved frequency resolution — especially valuable when signal length is limited. The covariance method provides the best resolution but requires care about stability.
